---
name: Image organization strategy
description: Blog images should be organized in per-post subdirectories under assets/images/posts/
type: feedback
---

Use per-post folders for images: `assets/images/posts/<post-slug>/image.png`.

**Why:** Keeps things organized as posts accumulate multiple images, avoids name collisions without relying on prefixes.

**How to apply:** When adding images for a blog post, create a subfolder matching the post slug (title portion of the filename, without the date) under `assets/images/posts/`.
